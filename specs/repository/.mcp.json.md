# .mcp.json

## 개요
MCP(Model Context Protocol) 서버 설정 파일입니다. 이 파일은:
- AI 개발 도구(Claude Code)와의 통합 설정
- Svelte 공식 MCP 서버 연결
- HTTP 프로토콜 기반 통신

## 소스 코드

```json
{
	"mcpServers": {
		"svelte": {
			"type": "http",
			"url": "https://mcp.svelte.dev/mcp"
		}
	}
}
```

## 주요 설정

### MCP 서버 목록 (mcpServers)

#### Svelte MCP 서버
- **이름**: `svelte`
- **type**: `http` - HTTP 프로토콜 사용
- **url**: `https://mcp.svelte.dev/mcp` - Svelte 공식 MCP 서버 엔드포인트

## MCP(Model Context Protocol)란?
MCP는 AI 개발 도구가 프로젝트 컨텍스트를 이해하고 더 나은 코드 지원을 제공하기 위한 프로토콜입니다.

### Svelte MCP 서버 기능
- Svelte/SvelteKit 코드 분석
- 컴포넌트 구조 이해
- Runes (Svelte 5) 문법 지원
- 라우팅 및 레이아웃 인식
- 타입 안전성 검증

## 사용 환경
- **Claude Code**: Anthropic의 공식 CLI 도구
- **VS Code 확장**: Claude Dev 등 MCP 지원 확장
- **기타 AI 도구**: MCP 프로토콜을 지원하는 모든 도구
