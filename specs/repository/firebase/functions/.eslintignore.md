# 목적
이 문서는 `firebase/functions/.eslintignore` 파일의 전체 내용을 기록하여 SED 스펙만으로도 Sonub 프로젝트를 재구성할 수 있도록 합니다.

## 파일 정보
- 상대 경로: `firebase/functions/.eslintignore`
- MIME: `text/plain`
- 유형: 텍스트

# 원본 소스 코드
```````text
# ESLint ignore 파일
# 테스트 파일은 프로덕션 코드가 아니므로 린팅에서 제외

# Test 폴더 전체 제외
test/**

# 빌드 결과물 제외
lib/**

# Node modules
node_modules/**

```````
